# MuevoCL - Servicios de Mudanzas

Plataforma web oficial para MuevoCL, enfocada en la prestación de servicios de transporte de carga, mudanzas locales e interregionales. El sitio permite a los clientes conocer el equipamiento, los servicios disponibles y gestionar cotizaciones de manera directa.

## Caracteristicas Principales

* Cotizaciones Directas: Integración con WhatsApp para presupuestos inmediatos y sin costo.
* Servicios Completos: Información detallada sobre mudanzas locales, interregionales y carga general.
* Información de Flota: Galería y especificaciones del camión (Mitsubishi Fuso cerrado de 27 metros cúbicos).
* Disponibilidad Total: Información sobre atención 24/7 y compromiso de puntualidad.
* Formulario de Contacto: Canal directo para consultas personalizadas.

## Tecnologias Utilizadas

* React
* Next.js
* Tailwind CSS

Si deseas ejecutar el proyecto de forma local, sigue estos pasos:

1. Clona el repositorio:
   git clone https://github.com/tu-usuario/nombre-del-repo.git

2. Instala las dependencias:
   npm install

3. Inicia el servidor de desarrollo:
   npm run dev
